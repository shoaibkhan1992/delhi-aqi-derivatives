Delhi AQI Composite Index (DAQI)

DAQI is a formally documented index constructed from CPCB-monitored stations across Delhi. It aggregates PM2.5, PM10, NO₂, and O₃ readings into a single tradeable reference value, using a weighted average methodology based on station coverage and population density.

Index Construction

Reference stationsAnand Vihar, RK Puram, Punjabi Bagh, Dwarka, ITO, Rohini, Lodhi Road
PollutantsPM2.5 (40%), PM10 (30%), NO₂ (20%), O₃ (10%)
CalculationWeighted arithmetic mean, daily 24-hr average
Data sourceCPCB / OpenAQ API (real-time)
Settlement basisCalendar-month average of daily DAQI readings

GRAP Trigger Levels

Delhi's Graded Response Action Plan creates discrete economic impacts at AQI thresholds — the natural anchor for option strike design.

Stage 0
AQI 0–200
Normal operations
Stage I
201–300
Light restrictions
Stage II–III
301–400
Construction halts, school closures
Stage IV
400+
Heavy industry shutdown, odd-even driving

Hedging rationale

  • Airlines: reduced demand / diversions when AQI triggers flight restrictions
  • Outdoor events / IPL: cancellation risk above Stage II
  • Construction firms: mandated halts → revenue loss proportional to AQI severity
  • Hospitality: hotel occupancy drops ~18% in high-AQI months (Oct–Jan)
  • Health insurers: claims spike correlated with AQI > 300
